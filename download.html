1. Styles

Compressed CSS files

Download zip file containing all the necessary .css files and include them in your web page in the head section as shown below:

<head>
 <!-- Styles -->
 <link href="css/bootstrap/bootstrap.css" rel="stylesheet">
 <link href="css/bootstrap/responsive.css" rel="stylesheet">
 <link href="css/simobil/simobil.css" rel="stylesheet">
 <link href="css/simobil/responsive.css" rel="stylesheet">
</head>

Less source files

Download zip file containing all the source .less files. General info on compiling can be found here.

Compile these files:

Source file: Output file:
less/bootstrap/bootstrap.less css/bootstrap/bootstrap.css
less/bootstrap/responsive.less css/bootstrap/responsive.css
less/simobils/simobil.less css/simobil/simobil.css
less/simobils/responsive.less css/simobil/responsive.css

2. Fonts

This zip file contains the font used on Si.mobil website.

CSS example of embeding the font:

@font-face {
 font-family:'SimbleWeb';
 src:
     url('path/to/fonts/Simble-Regular.eot?#iefix') format('embedded-opentype'),
     url('path/to/fonts/Simble-Regular.woff') format('woff'),
     url('path/to/fonts/Simble-Regular.ttf') format('truetype');
 font-weight: normal;
}
@font-face {
 font-family:'SimbleWeb';
 src:
     url('path/to/fonts/Simble-Bold.eot?#iefix') format('embedded-opentype'),
     url('path/to/fonts/Simble-Bold.woff') format('woff'),
     url('path/to/fonts/Simble-Bold.ttf') format('truetype');
 font-weight: bold;
}
@font-face {
 font-family:'SimbleWeb';
 src:
     url('path/to/fonts/Simble-Italic.eot?#iefix') format('embedded-opentype'),

     url('path/to/fonts/Simble-Italic.ttf') format('truetype');

}
